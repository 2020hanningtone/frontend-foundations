My Frontend Learning Journey

Starting with HTML

Everything begins with HTML. I learned how to structure pages using headings, paragraphs, lists, links, and images. This foundation made building webpages clear and fun.

Creating My First Page

I made a simple personal profile page with my name, bio, and a picture. It felt amazing to see my code come alive in the browser.

Moving on to CSS

After HTML, I learned CSS to style my pages. I discovered colors, fonts, margins, padding, flexbox, and grid layouts.

Styling My Profile Page

I added background colors, styled my headings, and made my page responsive. Seeing a beautiful design emerge from plain HTML was very satisfying.

Learning JavaScript

With HTML and CSS ready, I started JavaScript. I learned variables, functions, loops, and DOM manipulation.

Adding Interactivity

I made buttons that changed colors and displayed alerts. I realized JavaScript could make websites alive!

Finally, React

React allowed me to combine everything I learned. Components, props, state, and hooks became my new tools for building dynamic web applications.
